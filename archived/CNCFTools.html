1. App Definition and Development

  • Application Definition & Image Build: Helm, Artifact Hub
  • Continuous Integration & Delivery: Argo, GitLab, Jenkins, Spinnaker
  • Database: MySQL
  • Streaming & Messaging: No tools listed

2. Orchestration & Management

  • Scheduling & Orchestration: KEDA, Kubernetes, Knative, Kubeflow, Open Cluster Management, OpenFunction
  • Service Mesh: Istio, Linkerd, Consul

  • Coordination & Service Discovery: CoreDNS, ETCD

  • Container Runtime: CRI-O, Containerd, rkt, runc
  • Cloud Native Network: CNI, Cilium, Flannel, Kube-vip
  • Cloud Native Storage: CubeFS, Rook, Longhorn

4. Provisioning

  • Security & Compliance: cert-manager, KeyCLOAK
  • Container Registry: JFrog Artifactory, Dragonfly, Google Container Registry, Alibaba Cloud Container Registry, Amazon Elastic Container Registry, Azure Registry, IBM Cloud Container Registry
  • Automation & Configuration: Ansible, Terraform, KubeEdge, OpenStack, Puppet, Akri, Pulumi
  • Key Management: Vault, OAuth2 Proxy

5. Observability and Analysis

  • Observability: Fluentd, Jaeger, Prometheus, Grafana, Elastic, Datadog, Zabbix, Nagios, Splunk, Alibaba, Amazon
  • Chaos Engineering: ChaosMeta, Chaos Mesh, Litmus, Krkn, PowerfulSeal
  • Continuous Optimization: KubeCost, OpenCost
  • Feature Flagging: OpenFeature, FeatureHub, Bucket, DevCycle
